题目：打印出输入的字符。 
```C
#include <stdio.h>

int main () {
		int c;
		while ((c = getchar()) != EOF) {
				putchar(c);
				c = getchar();
		}
}

```
这个代码看起来似乎是没有问题，但是，实际上的输出将会变少。
输入：hello
输出：hlo

一开始是很另人感觉到疑惑的，我一度以为是语言的问题。因为，当我改为另一种方式的时候，运行又是正确的了。
```C
#include <stdio.h>

int main () {
		int c;
		c = getchar();
		while (c != EOF) {
				putchar(c);
				c = getchar();
		}
}
```
